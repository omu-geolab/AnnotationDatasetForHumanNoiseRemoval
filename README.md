Annotation Dataset For Human Noise Removal
--
### 概要 / Overview
異なる2つの現場で3次元計測した点群データに人間点群のみをアノテーションしたデータです。  
This is 3D measurement point cloud data from two different sites, annotated only with human point clouds.
